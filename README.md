n2pm
====

n2pm is NOT a Node Package Manager.

it's just a downloader/uploader ;)

```console
$ n2pm install <name> [<url> [--force]]
$ n2pm uninstall <name>
$ n2pm list
$ n2pm concat <name>... [--output <name>]
$ n2pm cleanup --force
```
